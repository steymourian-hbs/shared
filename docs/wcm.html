Authoring

Overlay

Add an ?overlay= parameter to the page to see an overlay

Or embed the overlays in the template and extract with overlay=1

Add an ?overlay=grid parameter to the page to see the grid

Grid Overlay

Press CTL+SHIFT to view the overlay grid

Clear Cache

Add an ?_hbs_cache_disabled=1 parameter to clear any page caching

Anonymous MasterPage

Add an ?_hbs_masterpage_anonymous=1 parameter to switch to the ribbon-free masterpage.

Edit Chevron
