Links that should open in a NEW tab

  1. Any non hbs.edu site
  2. This includes Harvard.edu
  3. ..and hbr.org

PDF links always open in a NEW tab. On hover, show the black icon above the link, at your cursor point.
If the link is an image, do not apply hover effect.

  1. An internal PDF
  2. An external PDF

Links that should automatically ignore conventions

  1. href values that contain "javascript:"
  2. href values that = "#"
  3. When in SharePoint edit mode
