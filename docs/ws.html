parameters:
  type - 
  limit - 
  searchquery - 

examples:
  http://www.hbs.edu/faculty/faculty-publications-rss.aspx?type=Cases&limit=10

Faculty Info


http://www.hbs.edu/faculty/faculty-info-rss.aspx


parameters
  names - Semicolon separated list of Full Names, 
          Full Names without Middle Initial, or Username
  
examples:
  http://www.hbs.edu/faculty/faculty-info-rss.aspx?names=michael+porter;lpaine;jalvarez;
