Type .type-framework


The HBS.edu typographic scale is built upon a ratio derived from print material and should be adhered to, not only to maintain a singular typographic voice and rhythm, but, perhaps more importantly, to ensure information hierarchy remains clear and easy for the user to understand. Unlike the print material, only one type family-Trade Gothic-is indicated for web use.

Values below are shown as font size (line-height). Line-height is based on the 8px baseline; where a class is set all caps, the line-height value is slim to enhance typographic tension and to closely mimic the printed material's design intent. If an additional size is needed, multiply the closest value by 1.38 and round to the nearest whole number.* For example, 61 x 1.38 = 84, 84 x 1.38 = 116, and so on.

Pushing Styles

Styles can be pushed to child tags using a convention like this .h3-beta. This will designate all h3 within the tag as a beta style.

Any combination of the above styles can be pushed to child tags. eg. .h1-alpha-uc .h2-gamma .blockquote-eta-uc .big-delta-uc .h5-eta-uc
