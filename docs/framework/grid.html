Each row is designated with a .row class. Use .row-fluid when you need a grid to fit a compressed container. 
<div class="row">
  <div class="span4">...</div>
  <div class="span8">...</div>
</div>

Use the .container class to set the 984px page width. 
<div class="container">
   ... page body goes here ...
</div>

Use the .vrule class to set a vertical line on the front edge of each column.

Note: you cannot put these on a .row due to the negative margin
